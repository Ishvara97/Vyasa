# Vedasya Rasam

This project is intended to parse Vedic Suktas and other Sanskrit Texts with the purposes of discovering quantitative linguistic patterns and paradigms for evaluations and comparisons. 

### Example

R.V. 10.125.4.b

**अमन्तवो मां त उप कषियन्ति शरुधिश्रुत शरद्धिवं ते वदामि ||**

_amantavo māṃ ta upa kṣiyanti **śrudhiśruta śraddhivaṃ te vadāmi** ||_

Within the bolded hemistich is contained:
- "the vowels _a_, _i_, _u_ and a single icon each of the oppositions of quantity (_a_ : _ā_) and nasalization (_a_ :_aṃ_)"
- "a single sibilant _ś_"
- "a single liquid _r_"
- "a single semi-vowel (glide) _v_"
- "a single nasal _m_"
- "and a single order of stops, _t_ _d_ _dh_ as tokens of the oppositions of voicing (_t_ : _d_) and aspiration or murmur (_d_ : _dh_)."[^1]
